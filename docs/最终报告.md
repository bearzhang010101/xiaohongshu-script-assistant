## 一、我是怎么找博主的

### 调研思路

围绕客户 Brief 的核心要求——轻食酸奶「轻醒」品牌、0蔗糖高蛋白希腊酸奶、目标 22-35 岁城市女性、自然种草不硬广——我在小红书上确定了博主的筛选标准：

1. 内容方向：健身、自律生活、轻食/早餐、vlog 日常
2. 粉丝画像：以 22-35 岁城市女性为主，关注健康生活方式
3. 内容调性：真实、生活化、有个人风格，非纯商业号
4. 商业匹配度：内容场景天然适合食品/饮品植入（早餐、运动后、下午茶）

### 10位博主调研

我筛选了 10 位符合上述条件的博主，逐一记录了主页链接、内容方向、粉丝画像和代表作品，详见 docs/01-博主调研报告.md。

### 最终选择：@静香

在 10 位博主中，我选择了 @静香，理由如下：

- 人设极其匹配：「6am 早起自律达人」，每天 6 点健身，天然创造了"早餐"和"运动后"两大产品植入场景，无需刻意安排
- 视觉公式可复用：提炼出「4段式高能自律视觉公式」——前3秒快闪 → 运动打卡 → 场景切换 → 美食收尾，结构清晰可模仿
- 表达方式高级：「自嘲式自律」，用"慢慢来 坚持就是胜利""奖励早起的自己"替代说教式种草，观众接受度高
- 商业价值好：11.95 万赞藏，中腰部成长型博主，粉丝画像极其精准，性价比高

## 二、我是怎么参考内容的

### 内容拆解方法

我系统分析了 @静香 近期 5-10 条视频，从以下维度进行拆解：

1. 视觉公式：前3秒 iPad 时间特写 + 运动装备快闪 + 镜前 Look → 健身房训练（冷色调）→ 居家场景（暖色调）→ 美食收尾
2. 文案风格：低姿态碎碎念，自嘲式自律，"慢慢来""奖励自己""感谢没有赖床的自己"
3. 镜头语言：主观 POV + 侧面全景 + 特写俯拍 + 镜前自拍，多角度切换
4. BGM 选择：强律动欧美说唱，《No Broke Boys》等
5. 植入方式：产品从不单独出场，始终作为自律生活的道具之一融入画面

### 双重预埋法

基于对博主内容的深度拆解，我设计了「双重预埋法」植入策略：

- 第1秒 潜意识层：快闪中产品作为运动装备前置亮相，观众尚未意识到广告
- 第22秒 功能层：练后早餐自然引出产品，高蛋白→运动修复
- 第26秒 体验层：快闪展示制作过程，色彩+食材搭配视觉种草
- 第30秒 情感层：冷静满足地品尝，自嘲式自律文案收束
- 第35秒 记忆层：空碗特写+阳光+淡出，点到为止的软植入

## 三、我是怎么设计脚本和分镜的

### 设计原则

1. 完全复刻博主视觉公式：不另起炉灶，直接套用 @静香 的 4 段式结构
2. 产品自然融入：酸奶作为运动装备、早餐食材出现，从不说"推荐""安利"
3. 合规先行：全片不出现"减肥""瘦身""降糖"等违规词，不夸大效果
4. 可真实拍摄：每个镜头都是博主日常会拍的，没有特效或摆拍

### 脚本结构（40秒）

前3秒快闪 → 运动打卡 → 场景切换 → 美食收尾

完整脚本和12镜头分镜设计详见 docs/03-短视频脚本与分镜.md。

## 四、合规风险检查

- 全片未出现"减肥、减脂、瘦身、降糖"等功效词汇
- 未使用"最、第一、100%"等绝对化用语
- 产品卖点客观表述：高蛋白、0蔗糖、饱腹感
- 植入全部融入博主原有视频公式，无硬广感

## 五、AI 工作流设计

5步Agent工作流（详见 docs/02-AI工作流设计.md）：

1. Brief 拆解 → 提取品牌、产品、卖点、人群、合规红线
2. 博主风格分析 → 拆解视觉公式、文案风格、表达方式
3. 脚本生成 → 基于风格画像产出完整脚本和分镜
4. 风险质检 → 多维度合规与风格匹配检查
5. 飞书写入 → 通过开放API自动创建文档并写入
