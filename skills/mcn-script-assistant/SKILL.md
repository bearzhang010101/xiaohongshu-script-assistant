# MCN Script Assistant - MCN商单脚本生成助手

## Skill 概述

一个面向MCN机构的AI自动化脚本生成工具，能够基于客户brief和目标博主风格，自动产出符合博主表达方式的种草短视频脚本，并完成合规质检和飞书文档同步。

---

## 适用场景

### 主要场景
- **MCN商单脚本快速产出**：接到品牌brief后，快速生成符合博主风格的初版脚本
- **博主风格化内容批量生成**：同一品牌对接多位博主时，批量生成各博主专属风格脚本
- **品牌种草视频脚本创作**：小红书/抖音等平台的种草短视频脚本创作
- **内容合规性自动化质检**：脚本产出后自动进行合规风险检查
- **飞书文档自动化交付**：脚本直接写入飞书文档，方便团队协作和客户审阅

### 适配行业
- 食品饮料（轻食、酸奶、健康零食等）
- 美妆护肤
- 运动健身
- 生活方式
- 快消品

## 输入材料

### 必需材料
1. **客户Brief文档**
   - 品牌名称、产品名称
   - 核心卖点（3-5个）
   - 目标人群描述
   - 投放平台要求
   - 合规红线说明

2. **目标博主信息**
   - 博主主页链接
   - 博主简介/人设描述
   - 3-5条代表作品标题或文案
   - 粉丝画像（可选）

3. **产品基础信息**
   - 产品图片/包装描述
   - 口味/规格
   - 使用场景建议

### 可选材料
- 参考脚本范例
- 品牌视觉规范
- 竞品投放案例

## 执行步骤

### Step 1: Brief结构化拆解
**目标**：将自然语言brief转化为结构化JSON数据

**输入**：客户brief文本
**输出**：结构化brief对象

**执行逻辑**：
- 提取品牌、产品、卖点、人群等核心字段
- 识别合规红线和禁忌表述
- 标准化数据格式，供后续Agent调用

### Step 2: 博主风格画像分析
**目标**：深度分析博主的内容风格、语言特点、表达方式

**输入**：博主主页信息 + 代表作品
**输出**：风格画像JSON

**分析维度**：
- 人设关键词（3-5个）
- 语言风格（口语化/专业/幽默等）
- 常用话术和口头禅
- 典型内容结构（钩子-主体-结尾）
- 常用钩子类型
- 偏好内容场景
- 适合的产品植入方式

### Step 3: 风格化脚本生成
**目标**：基于brief和风格画像，生成符合博主风格的完整脚本

**输入**：结构化brief + 风格画像
**输出**：完整短视频脚本

**脚本包含**：
- 标题（小红书风格，带钩子）
- 基本信息（时长、场景、风格）
- 完整字幕文案
- 分镜设计表（时间戳、景别/运镜、画面视觉内容、字幕文案、设计意图）
- 产品植入点标注
- 合规风险提醒

### Step 4: 多维度风险质检
**目标**：检查脚本合规性、风格匹配度、内容质量

**输入**：生成的脚本
**输出**：质检报告 + 修改建议

**检查维度**：
1. **合规检查**：是否触碰广告法、平台规则红线
2. **风格检查**：是否符合博主真实表达方式
3. **植入检查**：产品植入是否自然，有无硬广感
4. **逻辑检查**：内容是否通顺合理
5. **平台适配**：是否符合目标平台调性

### Step 5: 飞书文档自动输出
**目标**：将最终脚本自动写入飞书文档，生成可分享链接

**输入**：最终脚本内容
**输出**：飞书在线文档链接

**实现方式**：
- 调用飞书开放API创建文档
- 按格式写入标题、正文、表格
- 返回文档链接

## 输出格式

### 标准输出物

#### 1. 短视频脚本（Markdown格式）
```markdown
# [脚本标题]

## 基本信息
- 时长：xx秒
- 场景：健身房→独居房间（如@静香 练后早餐场景）
- 风格：自律vlog风，自嘲式碎碎念（如@静香）

## 字幕文案
[完整字幕内容]

## 分镜设计
| 时间戳 | 景别/运镜 | 画面视觉内容 | 画面花体字/字幕文案 | 设计意图与调性把控 |
|------|------|----------|-----------|------|

## 产品植入点
1. [时间点] - [植入方式]

## 合规风险提醒
- [检查项]
```

#### 2. 合规质检报告（JSON格式）
```json
{
  "pass": true,
  "compliance_issues": [],
  "style_issues": [],
  "suggestions": [],
  "risk_level": "low"
}
```

#### 3. 飞书文档链接
- 在线可访问的飞书文档URL
- 支持团队协作编辑

## 风险检查清单

### 合规风险（必须全部通过）
- [ ] 无减肥、减脂、瘦身、减重等功效承诺
- [ ] 无降糖、治病、疗效等医疗功效宣称
- [ ] 无"最"、"第一"、"100%"、"绝无仅有"等绝对化用语
- [ ] 无保健功能、疾病预防等违规表述
- [ ] 无虚假夸大的效果描述
- [ ] 产品卖点表述客观真实
- [ ] 符合广告法和平台内容规范

### 内容质量
- [ ] 符合博主真实的语言风格和表达方式
- [ ] 使用博主常用的话术和口头禅
- [ ] 产品植入自然，无硬广感
- [ ] 场景符合博主日常内容定位
- [ ] 开头有钩子，能吸引用户停留
- [ ] 信息密度适中，符合短视频节奏
- [ ] 结尾有互动引导

### 品牌适配
- [ ] 核心卖点准确传达
- [ ] 品牌名称和产品名称正确
- [ ] 目标人群匹配
- [ ] 品牌调性一致
- [ ] 无负面联想风险

### 平台适配
- [ ] 符合小红书内容调性
- [ ] 标题带有关键词和情绪点
- [ ] 时长符合短视频推荐（30-60秒）
- [ ] 画面描述可执行
- [ ] 字幕信息清晰

## 使用示例

### 输入示例
```
品牌：轻醒
产品：0蔗糖高蛋白希腊酸奶
卖点：高蛋白、饱腹感、低负担
目标人群：22-35岁城市女性，关注健身控糖
博主：@静香（6am早起自律Vlog）
```

### 输出示例
生成一条40秒的自律vlog种草脚本，完全复刻博主4段式高能自律视觉公式，通过双重预埋法实现潜意识层+功能层双层次植入，使用"慢慢来 坚持就是胜利"、"奖励早起的自己"、"感谢今天也没有赖床的自己"等博主标志性语言，自然植入产品，通过合规质检后写入飞书文档。
